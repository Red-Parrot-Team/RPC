# RPChat
secure chat (Python)

## Установка
1) Установать [Python](https://www.python.org/downloads/) с оф. сайта
2) Попробовать запустить `client.py`:
```
python3 client.py
```
3) При ошибке связанной с tkinter выполнить комманду:
```bash
sudo apt-get update
sudo apt-get install python3-tk
```

4) Если помогло, то:

- Сервер запускается с помощью комманды `python3 server.py`
- Клиент коммандой `python3 client.py`

Прекращаем работу сервера комбинацией Ctrl+C, А Клиента вводом комманды `/exit`
